Normally, if no topological contribution, we have:
$$\begin{align}
S_{xy} & = \alpha H + \beta M(H)
\end{align}$$
Then if we do a linear fit in the saturation region of $M$, we have:
$$\begin{align}
aH+b= \alpha H+ \beta M_{sat}
\end{align}$$
Then the intercept is:
$$b= \beta M_{sat}$$
If the ratio $\frac{b}{M_{sat}}$ is fixed, we know that there is no topological contribution.
