
>[! Note] Definition 1.1
>Let $I=[0,1]$, $X$ be a topological space. A path $\alpha$ is a continuous function $\alpha:I\rightarrow X$. If $\alpha(0)=\alpha(1)=x_{0}$, we say $\alpha$ is a loop with base point $x_{0}$.

我们考虑path上的代数结构。我们定义path间的二元运算：

>[!Note] Definition 1.2
>Let $\alpha,\beta$ be paths such that $\alpha(1)=\beta(0)$. Define the product between $\alpha,\beta$ as:
>$$\alpha*\beta(s)= \left\{\begin{align}
 & \alpha(2s),\ 0\leq s \leq \frac{1}{2} \\
 & \beta(2s-1),\ \frac{1}{2}\leq s \leq 1
\end{align}\right.$$

显然，由于$\alpha(1)=\beta(0)$，$\alpha*\beta$就还是连续的，仍然是一条path。再定义path的逆：

>[!Note] Definition 1.3
>Let $\alpha$ be a path. Define the inverse of $\alpha$ as:
>$$\alpha ^{-1}(s)=\alpha(1-s)$$
## Ex:

令$\alpha$为一个path。那么$\alpha*\alpha ^{-1}$就是先走完$\alpha$，然后有从末端往回走返回起点。
