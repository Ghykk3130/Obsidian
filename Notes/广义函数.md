
我们考虑从$\mathbb{R}\rightarrow \mathbb{C}$的函数。我们先定义测试函数空间。

>[!Note] Definition 1
>Define the space of test functions as:
>$$\mathcal{D}(\mathbb{R})=\{ \phi\in C^{\infty}(\mathbb{R})|\phi\text{ is supported on some compact set} \}$$

考虑一个函数列$g_{n}\in\mathcal{D}(\mathbb{R})$。我们定义$\mathcal{D}(\mathbb{R})$中的收敛：

>[!Note] Definition 2
>Let $g_{n},g\in\mathcal{D}(\mathbb{R})$. We say $g_{n}\rightarrow g$ in $\mathcal{D}(\mathbb{R})$ if there exists a compact set $K\subset \mathbb{R}$ such that  
>$$\text{sup}_{x\in K} |g_{n}^{m}(x)-g^{(m)}(x)|\rightarrow 0,\ \forall m \in \mathbb{N} $$

接下来定义广义函数：

>[!Note] Definition 3
>Let $T:\mathcal{D}(\mathbb{R})\rightarrow \mathbb{C}$ be a linear functional. We say $T$ is a generalized function if given a sequence $\phi_{n}\in \mathcal{D}(\mathbb{R}),\ \phi_{n}\rightarrow \phi$, we have $\langle T,\phi_{n}\rangle \rightarrow \langle T,\phi\rangle$.
>Define the space of all generalized functions as $\mathcal{D}^{'}(\mathbb{R})$.

显然$\mathcal{D}^{'}(\mathbb{R})$是$\mathcal{D}(\mathbb{R})$的对偶空间。

回忆起局部$L^{1}$空间的定义。其中的函数不要求在$\mathbb{R}$上可积，只需要在任意紧集上可积即可。即：
$$L^{1}_{\text{loc}}(\mathbb{R})=\left\{  f:\mathbb{R}\rightarrow \mathbb{C}|f\text{ measurable, }\int_{K}dx|f|<\infty\ \forall K \subset \mathbb{R}\text{ compact}  \right\}$$
可以证明，给定$f\in L^{1}_{\text{loc}}(\mathbb{R})$，它可以诱导出一个广义函数：
$$\begin{align}
\langle T_{f},g\rangle & = \int_{-\infty}^{\infty}dx f(x)g(x),\ \forall g\in\mathcal{D}(\mathbb{R})
\end{align}$$
还可以证明，若$T_{f}=T_{h}$，那么$f=h\text{ a.e.}$。

对于$f(x)= \frac{1}{x}$，显然在$0$附近不可积。但是它还是可以引导出一个广义函数。我们定义：

>[!Note] Definition 4
>Given $f(x)= \frac{1}{x}$, define its Cauchy principal value $\mathcal{P} \frac{1}{x}\in\mathcal{D}^{'}(\mathbb{R})$ such that:
>$$\begin{align}
\left\langle  \mathcal{P} \frac{1}{x},g \right\rangle= \lim_{ \epsilon \to 0^{+} } \int_{|x|> \epsilon} dx \frac{1}{x} g(x)
\end{align},\ \forall g\in \mathcal{D}(\mathbb{R})$$

我们还可以定义所谓的Dirac delta。

>[!Note] Definition 5
>Define the Dirac delta function $\delta \in \mathcal{D}^{'}(\mathbb{R})$ such that:
>$$\langle \delta,g\rangle= g(0),\ \forall g\in\mathcal{D}$$
