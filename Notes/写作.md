# 1. 构造观点的方法

**分点构造**
五个角度构造观点：
- 个人层面
- 社会层面
- 效率层面：是否提高了产出？
- 长期层面：短期的帮助怎么样？长期的帮助怎么样？
- 让步层面：是否在某些情况有害

比如题目：
“学生是否应该花更多时间学习历史？”
你可以迅速生成：
- 个人：学习历史能提升判断力，因为人能从过去的错误中吸取经验。
- 社会：历史教育能增强公民意识，让人理解制度、战争和社会冲突的来源。
- 长期：短期看历史不如技能课实用，但长期能帮助人形成更成熟的世界观。
- 让步：如果只是背日期和人物，历史课会低效；关键是要联系现实问题。

**按对象构造**
很多题目想不出点子，是因为只盯着抽象观点。你可以立刻问：“这个事情影响了哪些人？”

常见主体：
- 个人层面：学生，家长，老师，员工
- 社会层面：公司，政府，市场，消费者，社区，弱势群体，未来一代

例如题目：“公司是否应该允许员工在家办公？”
按主体拆：
- 员工：节省通勤时间，工作生活更平衡。
- 公司：减少办公室成本，但管理和沟通变难。
- 家庭：父母能更灵活照顾孩子。
- 社会：减少交通拥堵和污染。
- 新员工：可能难以融入团队、学习公司文化。

GRE中，加上带有条件的让步最好。一般情况下，我支持，但是在某些特殊情况下，我不支持。文章划分成同意和让步两部分。

“政府是否应该资助艺术？”
快速构造：
- 艺术从业者：鼓励艺术创造（融入后两点）
- 市场：艺术有公共价值，构成一种艺术的经济模式。
- 社会：艺术能保存文化身份，提升社会凝聚力。
- 让步：如果过于抽象，市场未必能充分支持。有的行为艺术甚至污染环境，堵塞交通。不支持。
- 让步：政府资助不应挤占医疗、教育等基本预算。
- 条件：资助应透明，并优先支持有公共教育价值的项目。


**常见观点：**

你可以记一些高频主题的“常用理由”。

**教育类**

- 培养批判性思维，而不是只记知识。
- 提高就业竞争力。
- 帮助学生发现兴趣。
- 促进公平，给低收入学生更多机会。
- 过度压力可能损害心理健康和创造力。

**科技类**

- 提高效率，降低成本。
- 扩大信息获取机会。
- 改变沟通和学习方式。
- 可能导致隐私问题、依赖、注意力下降。
- 技术本身中立，关键在使用方式和监管。

**政府类**

- 政府能提供公共产品，如教育、交通、医疗、环保。
- 政府需要保护弱势群体。
- 过度干预可能降低效率或限制自由。
- 政策应考虑成本、透明度和长期影响。

**工作类**

- 高薪能提高生活质量，但不一定带来满足感。
- 灵活工作能提高幸福感和效率。
- 团队合作促进创新。
- 竞争能激励进步，但过度竞争会造成压力。
- 工作意义、成长空间和人际关系也很重要。

**社会 / 文化类**

- 传统能提供身份认同和社会稳定。
- 创新推动社会进步。
- 多元文化能扩大视野。
- 文化保护和经济发展可能冲突。
- 社会规范有时需要随着时代改变。

# 2. 每个观点段内逻辑的构造

想出一个点后，不要停在一句话。用这个链条扩展：观点 → 原因 → 结果 → 例子 → 对比

例如某段要写“大学生应该参加实习。”
- 观点：实习能帮助学生做职业选择。
- 原因：课堂知识通常比较抽象，学生不知道真实工作是什么样。
- 结果：通过实习，学生可以发现自己是否适合某个行业。
- 例子：一个学金融的学生在投行实习后，可能发现自己更喜欢数据分析而不是客户沟通。
- 让步：如果没有实习，他可能毕业后才发现职业不匹配，浪费更多时间。

# 3. 实践方法

看到题目后这样做：
1. **改写题目**：它到底在问什么选择？
2. **站队**：先选一个更容易写的立场，不一定是你真实想法。
3. **列主体**：谁受影响？
4. **列利弊**：至少写 2 个支持点 + 1 个反方点。
5. **加条件**：什么时候成立？什么时候不成立？
6. **选最容易举例的 2–3 个点写正文。**
