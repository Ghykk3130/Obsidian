# 1.1 流形的拓扑结构和微分结构

>[!Note] Definition 1.1.1
>Let $M,N$ be topological space. Then $f:M\rightarrow N$ is a homeomorphism if $f,f^{-1}$ are continuous and one to one.

>[!Note] Definition 1.1.2
>Let $M$ be a topological space. We say $M$ is Hausdorff if given $a,b\in M,\ a\neq b$, there exists $U_{a},U_{b}$ open such that $a\in U_{a},\ b\in U_{b},\ U_{a}\cap U_{b}=\phi$.

>[!Note] Definition 1.1.3
>Let $M$ be a topological space. We say $M$ is a manifold if:
>- $M$ is Hausdorff.
>- $\forall p\in M$, $\exists U\text{ open s.t. }p\in U$, and $U$ is homeomorphic to an open set in $\mathbb{R}^n(\mathbb{C}^n)$.

## Ex:
显然单位圆$S^1$每点邻域与$\mathbb{R}$同胚，且Hausdorff。所以为一维流形。

## Ex:
![[Pasted image 20260113144638.png|center|300]]
这些都不是流形。因为在若和$\mathbb{R}^n$同胚，则交点$p$处作小邻域$U$可以映到$\mathbb{R}^n$中开集$f(U)$。那么考虑$U \setminus\{ p \}$，相应地有$f(U\setminus \{ p \})=f(U)\setminus f(p)$。

回忆：一个流形若可以被两个开集分开，则成为seperable。若不行则称为connected。容易证明同胚保持connected part的数量。因为$U\setminus \{ p \}$有4或3个connected parts，而$f(U)\setminus f(p)$有两个或零个connected parts，所以同胚不可能存在。

>[!Note] Definition 1.1.4
>Let $\mathscr{A}=\{ U_{\alpha},\phi_{\alpha} \}$ be an atlas over manifold $M$. We say that $\mathscr{A}$ is a $C^k$ differential structure of $M$ if:
>- $\{ U_{\alpha} \}$ is an open cover of $M$.
>- Charts in $\mathscr{A}$ are $C^k$ compatible.
>- $\mathscr{A}$ is the largest atlas, i.e. if $(U,\phi)$ is compatible with $\forall(U_{\alpha},\phi_{\alpha})\in\mathscr{A}$, then $(U,\phi)\in \mathscr{A}$.

# 1.2 Tangent space

>[!Note] Definition 1.2.1
>Given $p\in M$. Let $x:\mathbb{R}\rightarrow M,\ t\mapsto x(t)$ be a curve passing through $p$ at $t=0$. Let $f\in C^{\infty}(M)$. Then define the tangent vector: $X_{p}:C^{\infty}(M)\rightarrow \mathbb{R}$ by:
>$$X_{p}(f)= \frac{d}{dt}|_{t=0}f(x(t))$$

>[!Note] Proposition 1.2.1
>$\{ X_{p}|x(t)\text{ passing through }p\text{ at }0 \}$ forms a vector space.
## Proof.

任取两条穿过$p$曲线$x(t),y(t)$。试图证明$aX_{p}+Y_{p}=Z_{p}$ for some curve $z(t)$。

我们发现，若在$p$点选取chart作为局部坐标，有：
$$X_{p}= \frac{d}{dt}= \frac{dx^i}{dt} \frac{\partial}{\partial x^i},\ Y_{p}= \frac{d}{dt}=\frac{dy^i}{dt} \frac{\partial}{\partial x^i}$$
考虑在坐标表示下定义：
$$z^i(t)=p^i+ \left( a \frac{dx^i}{dt} + \frac{dy^i}{dt}\right)t$$
则：
$Z_{p}= \left( a \frac{dx^i}{dt}+ \frac{dy^i}{dt} \right)\partial_{i}=aX_{p}+Y_{p}$
>[!Right]
>$\blacksquare$
